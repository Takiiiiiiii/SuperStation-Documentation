# SuperStation-Documentation
## Getting Started

If you have just received a SuperStationᵒⁿᵉ, and this is your first MiSTer based device, then welcome to the MiSTer Ecosystem! There are many things to learn about how MiSTer works as a platform, and the resources online are numerous. If you need an exhaustive reference to what the project entails, please review the subjects covered [here](https://github.com/MiSTer-devel/Wiki_MiSTer/wiki). You will find a lot of basic information about functionality of the project and expectations when using MiSTer based devices.

Additionally, if you are looking for an active community of SuperStationᵒⁿᵉ and MiSTer Pi users, you may also join the [Official Discord Channel](https://discord.com/invite/2EYTb8N) or read related topics on the [Official Retro Remake Reddit](https://www.reddit.com/r/RetroRemake/).
